HTTP and HTTPS:

A browser sends in an HTTP request to the webserver, the web server returns the HTTP response to the web browser client (this is how the client is able to see the website). HTTP follows the standard client-server model or request response model.

The HTTPS (hypertext transfer protocol secure) in web-browser indicates that your communication with the server is encrypted meaning security of data transfer. The only difference between these two protocols (HTTP and HTTPS) is that one is secure(HTTPS) while the other isn't (HTTP)
